Программа "Measurelook" разработана Речкаловым Тимофеем по личной инициативе.

Программа "Measurelook" предназначена для визуалиции результатов вычислительных экспериментов для файлов фиксированного формата. Для получения большей информации см. репозиторий проекта.

Сайт автора: trechkalov.com

Программа бесплатна. Исходный код доступен в репозитории под лицензией Apache 2.

Авторы используемых иконок с сайта www.flaticon.com: Anton Saputro, Pavel Kozlov, Budi Tanrim, Catalin Fertu, Picol и Freepik. Автор иконки сортировки FatCow.

История версий
  • Версия 0.2.1 (7 May 2017) - мелкие улучшения.
  • Версия 0.2.0 (2 Apr 2017) - улучшение формата хранения эксперимента и графики по медоидам.
  • Версия 0.1.0 (20 Feb 2017) - первый закрытый релиз.
